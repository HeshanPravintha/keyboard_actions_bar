# Changelog

## 1.0.2

- Fixed deprecated `withOpacity` usage — replaced with `withValues()` to avoid precision loss.
- Updated package description to meet pub.dev length guidelines.

## 1.0.1

- Added demo video to showcase the package in action.

## 1.0.0

First official release of **keyboard_actions_bar**.

A fully customizable keyboard toolbar for Flutter with zero FocusNode boilerplate,
Material 3 theming, tap-outside dismiss, custom keyboard panels, and 6 built-in
actions — Done, Next, Prev, Clear, InsertAt, and Custom.
